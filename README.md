# Kkrypt0nn Tap

## How do I install these formulae?

`brew install kkrypt0nn/tap/<formula>`

Or `brew tap kkrypt0nn/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kkrypt0nn/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
